Nosotros somos tres

Tienes que hacer clone con el enlace que se genera Antes de nada, hacemos git pull de la master Creamos rama para la tarea: git checkout –b “Nombre-tarea” Hacemos la tarea Git add . Git commit –m “Nombre del commit” Git push Copiamos el comando que nos da y metemos las credenciales si las pide Vamos a GitHub Vamos a branches Vamos a la rama y pulsamos New pull request Créate pull request Pasamos la url a 2 compañeros para que vean los cambios Los compañeros irán a Files changed y podrán aprobar los cambios pulsando “Viewed” Botón verde Review changes -> approve Cuando tengas los aprovados de los compañeros podremos hacer un Merge Merge pull request Git checkout master Git pull para actualizar la rama master.
